# CareerOS AI

CareerOS AI is a Next.js career platform that helps users analyze resumes, match jobs, prepare for interviews, build portfolios, and get AI-powered career guidance.

## Features

- AI resume analysis and scoring
- Resume tailoring for job descriptions
- Smart job search tools
- Interview practice assistant
- AI portfolio generator
- Career dashboard
- Supabase-ready profile storage
- Modern responsive UI with Tailwind CSS

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- OpenAI API
- Supabase
- Vercel-ready deployment
